# Simple Anti-Cheat for Unity Projects
Модульная и легко расширяемая анти-чит система для игр на Unity, созданная для обнаружения признаков читерства игроков. Разработана для обеспечения базовой защиты игровых механик и предотвращения манипуляций с памятью.
![anti cheat ui](https://github.com/1D4n3/Anti-cheat-system/blob/main/Pictures/anti%20cheat%20ui.png)

## Особенности системы
- **Модульная архитектура.** Позволяет масштабировать систему под каждый игровой проект индивидуально и подключать сторонние модули отслеживания.
- **Централизованное управление.** Единая точка входа на базе паттерна Singleton, координирующая инициализацию, жизненный цикл модулей и обработку нарушений.
- **Гибкая конфигурация.** Настройки порогов чувствительности и интервалов проверок осуществляется напрямую в Unity.
- **Накопительная система подозрений.** Сводит к минимуму ложные срабатывания анти-чит системы.

## Интеграция анти-чит системы
 - В окне Project нажмите ПКМ -> Create -> AntiCheat -> Config
 - Назовите файл AntiCheatConfig и поместите его в папку Assets/Resources/ (чтобы ядро могло загрузить его автоматически при старте)
 - Настройте параметры чувствительности в инспекторе (максимально разрешенную скорость, интервалы проверок и реакцию на обнаружение и т.д.)

## Системные требоваия
- **Unity:** версия 6000.0.0f1 и выше
- **Платформы:** Windows, macOS, Android
